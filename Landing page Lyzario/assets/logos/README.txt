Logos du bandeau outils – Lyzario
=================================

Ajoute dans ce dossier un fichier pour chaque outil (SVG ou PNG, de préférence fond transparent).

Noms de fichiers attendus :
  - airtable.svg   (ou airtable.png)
  - make.svg       (Make / Integromat)
  - n8n.svg
  - chatgpt.svg    (OpenAI / ChatGPT)
  - notion.svg
  - google-drive.svg
  - slack.svg

Si tu utilises des PNG, renomme-les avec l’extension .svg dans le code (index.html) ou renomme le fichier en .svg.

Tu peux récupérer les logos officiels sur les sites des marques (pages « Brand », « Press », « Ressources ») ou via des banques d’icônes (avec droits d’usage).
